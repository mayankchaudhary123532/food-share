# 🍲 FoodShare – Food Donation Web Application

🚀 A full-stack web application designed to reduce food wastage by connecting donors (restaurants/individuals) with NGOs in real-time.

---

## 📌 Overview

FoodShare is a platform where:

* Donors can list surplus food
* NGOs can view and request available food
* Helps in efficient food distribution and minimizing waste

---

## ✨ Features

* 🥗 Donate surplus food easily
* 🏢 NGO dashboard to request food
* 🔄 Real-time food availability updates
* 📦 Organized food listings
* 🔐 Secure backend with environment variables
* 🌐 Responsive frontend UI

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/mayankchaudhary123532/food-share.git
cd food-share

---

### 2️⃣ Setup Backend

cd backend
npm install

Create a `.env` file and add:
MONGO_URI=your_mongodb_connection_string
PORT=5000

Run backend:
npm start

---

### 3️⃣ Setup Frontend

cd frontend

Open `index.html` in browser

---

## 🚀 Usage

1. Donor adds food details
2. Data is stored in MongoDB Atlas
3. NGOs can view and request food
4. Helps reduce food wastage efficiently
